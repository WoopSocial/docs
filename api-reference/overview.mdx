## Base URL

All API requests should be made to:

```
https://api.woopsocial.com/v1
```

## Authentication

You will need an API key to authenticate your requests. You can acquire one from the [WoopSocial dashboard](https://app.woopsocial.com/) by clicking on the "API" menu item on the left.

All endpoints should include the API key in the `Authorization` header:

```bash
Authorization: Bearer YOUR_API_KEY
```
