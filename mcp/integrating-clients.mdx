# Integrating MCP clients

Use this page to connect the WoopSocial MCP server to a client that supports streamable HTTP MCP.

## Prerequisites

- A WoopSocial account
- Access to the client you want to configure

## Claude Code CLI

Use the `claude mcp add` command to register WoopSocial with Claude Code.

```bash
claude mcp add --transport http woopsocial "https://api.woopsocial.com/mcp"
```

1. Ensure that you have a [WoopSocial account](https://app.woopsocial.com) and you are logged in
2. Start `claude`
3. In Claude, type `/mcp`
4. Select the `woopsocial · △ needs authentication` menu item
5. Select `Authenticate`
6. This should take you to WoopSocial where you can Authorize the MCP connection. Click the "Authorize" button.
7. Return to Claude
8. Ask Claude to do something with the MCP. For example tell it to "List my social accounts in woopsocial"

## Codex CLI

Use the `codex mcp add` command to register WoopSocial with Codex.

```bash
codex mcp add woopsocial --url "https://api.woopsocial.com/mcp"
```

## Claude Desktop

1. Open Claude settings.

<img src="/images/social-media-mcp/Claude%20Social%20Media%20MCP%20setup%20image%201.png" alt="Open Claude settings" />

2. Open the `Connectors` tab.

<img src="/images/social-media-mcp/Claude%20Social%20Media%20MCP%20setup%20image%202.png" alt="Open the Connectors tab in Claude" />

3. Select `Add custom connector`.

<img src="/images/social-media-mcp/Claude%20Social%20Media%20MCP%20setup%20image%203.png" alt="Add a custom connector in Claude" />

4. Enter a connector name and paste the MCP URL.

<img src="/images/social-media-mcp/Claude%20Social%20Media%20MCP%20setup%20image%204.png" alt="Enter the connector name and MCP URL" />

5. Save the connector.
6. Test the connection by asking Claude to list your social accounts.

<img src="/images/social-media-mcp/Claude%20Social%20Media%20MCP%20setup%20image%205.png" alt="Test the Claude connector by listing social accounts" />

If Claude blocks media uploads, add `*.cloudflarestorage.com` to the allowlist in its settings.

## ChatGPT

1. Open `Settings`.

<img src="/images/social-media-mcp/ChatGPT%20Social%20Media%20MCP%20setup%20image%201.png" alt="Open ChatGPT settings" />

2. Open `Apps`.
3. Enable `Developer Mode` if it is not already enabled.

<img src="/images/social-media-mcp/ChatGPT%20Social%20Media%20MCP%20setup%20image%202.png" alt="Enable Developer Mode in ChatGPT" />

4. Select `Create App`.
5. Enter these values:

- Name: `WoopSocial`
- Description: `Publish social media content`
- MCP Server URL: paste the MCP URL you copied earlier
- Authentication: `No Auth`

<img src="/images/social-media-mcp/ChatGPT%20Social%20Media%20MCP%20setup%20image%203.png" alt="Create the WoopSocial app in ChatGPT" />

6. Save the app.
7. Start a new chat and test the connection by asking ChatGPT to list your social accounts.

<img src="/images/social-media-mcp/ChatGPT%20Social%20Media%20MCP%20setup%20image%204.png" alt="Test the ChatGPT app by listing social accounts" />

# Note

If the client has an allowlist for outbound domains, add `*.cloudflarestorage.com` so media uploads work.
